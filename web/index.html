Get Started

This service allows you to download your school's menu calendar directly to your device using our simple menu form. For automation, you can also use a Siri shortcut or make an API call. Right now, Linq Connect is the only supported platform. There may be other platforms supported in the future.

How to Use the Siri Shortcut

  1. Install the Siri shortcut
  2. OPTIONAL: Add the ICS to Calendar shortcut
  3. Run the shortcut and provide the following information:
    • Building ID
    • District ID
    • Start Date (MM-DD-YYYY)
    • End Date (MM-DD-YYYY)
  4. The shortcut will generate and download the ICS file for you

More information on how to get the building and district IDs can be found here.

API Endpoint

For the more technical folk out there, you can use the API to generate the ICS file:

curl -s -X POST -d "buildingId=1234&districtId=5678&startDate=10-01-2024&endDate=10-31-2024" https://schoolmenuconnector.com/get-menu -o school_menu.ics

Note: Only POSTs are supported for the endpoint.

Parameters (send as form data):

  • buildingId
  • districtId
  • startDate
  • endDate
